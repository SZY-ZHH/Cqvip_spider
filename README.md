# 维普期刊爬虫（单位IP/免费下载版）

基于 DrissionPage 的维普期刊文献自动下载工具，支持单位IP直连和免费下载两种模式。

## 功能特性

- 🔍 按关键词搜索维普期刊文章
- 📥 自动下载PDF/CAJ全文（单位IP有权限即可下载）
- 🚫 自动识别并跳过需付费文章
- 📊 生成CSV搜索结果和总结报告
- 💾 断点续爬，中断后可继续
- 📁 按关键词自动创建文件夹

## 环境依赖

### Python版本

- Python 3.8+

### 安装依赖

```bash
pip install DrissionPage requests beautifulsoup4 lxml urllib3
```

### 浏览器要求

- 需要安装 Microsoft Edge 或 Google Chrome 浏览器
- 程序会自动检测浏览器路径

## 使用方法

### 方式1：交互模式（推荐）

```bash
python Cqvip.py
```

按提示输入：
```
请输入搜索关键词: 深度学习
请输入爬取数量(回车表示全部): 50
请输入输出目录(回车使用默认目录):
```

### 方式2：命令行模式

```bash
# 搜索关键词，下载全部
python Cqvip.py -k "深度学习"

# 指定下载数量
python Cqvip.py -k "深度学习" -n 50

# 指定输出目录
python Cqvip.py -k "深度学习" -n 50 -o "D:\论文"

# 批量模式（依次爬取预设关键词列表）
python Cqvip.py -b -n 20
```

### 参数说明

| 参数 | 缩写 | 说明 |
|------|------|------|
| `--keyword` | `-k` | 搜索关键词 |
| `--num` | `-n` | 爬取数量，0表示全部 |
| `--output` | `-o` | 输出目录 |
| `--batch` | `-b` | 批量模式 |

## 输出目录结构

```
输出目录/
├── 关键词/
│   ├── 关键词_维普搜索结果.csv    # 增量搜索结果
│   ├── 关键词_总结报告.csv        # 最终总结报告（含文件大小）
│   ├── 关键词_cqvip_progress.json # 断点续爬进度
│   └── articles/                  # 下载的PDF/CAJ文件
│       ├── 文章标题1.pdf
│       ├── 文章标题2.pdf
│       └── ...
```

## 下载逻辑

```
打开文章详情页
    ↓
查找下载按钮（免费下载 或 下载PDF）
    ↓ 找到
点击下载按钮
    ↓
等待3秒检查结果
    ├── 跳转到支付页面 → 关闭标签页，标记"需付费跳过"
    ├── 跳转到错误页面 → 标记"下载失败"
    └── 正常下载 → 等待文件下载完成 → 重命名文件
    ↓ 找不到按钮
标记"未找到下载按钮"，跳过
```

## 可配置参数

在 `Cqvip.py` 文件顶部可修改以下参数：

### 延迟与限流

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MIN_DELAY` | 0.5 | 请求间最小延迟（秒） |
| `MAX_DELAY` | 1.5 | 请求间最大延迟（秒） |
| `RATE_LIMIT_THRESHOLD` | 60 | 每N次请求触发长延迟 |
| `RATE_LIMIT_DELAY_MIN` | 10 | 长延迟最小时长（秒） |
| `RATE_LIMIT_DELAY_MAX` | 20 | 长延迟最大时长（秒） |

### 下载控制

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `DOWNLOAD_MAX_RETRY` | 2 | 下载按钮点击最大重试次数 |
| `DOWNLOAD_WAIT_TIMEOUT` | 60 | 常规文件下载等待超时（秒） |
| `LARGE_FILE_WAIT_TIMEOUT` | 120 | 大文件下载额外等待超时（秒） |

### 其他

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MAX_RETRY` | 5 | 网络请求最大重试次数 |
| `BATCH_SAVE_INTERVAL` | 10 | 每N篇自动保存CSV |
| `LOG_MAX_SIZE` | 5MB | 日志文件最大大小 |

## 单位IP用户说明

如果您通过单位IP直连维普（无需登录即可下载PDF）：

1. 直接运行程序即可，不需要登录
2. 程序会自动打开浏览器，等待您确认登录状态
3. 如果浏览器已自动登录，直接按 Enter 键确认
4. 程序会尝试点击所有"下载PDF"按钮，自动跳过需付费的文章

## 常见问题

### 1. 所有文章都显示"需付费跳过"

**原因**：浏览器未登录或单位IP未生效

**解决**：
- 在浏览器中手动登录维普账号
- 确认单位网络连接正常
- 检查浏览器是否能手动下载PDF

### 2. 下载失败，未检测到下载文件

**原因**：浏览器下载路径未正确设置

**解决**：
- 检查浏览器默认下载路径
- 关闭浏览器"下载前询问保存位置"选项
- 尝试手动下载一篇确认路径

### 3. 搜索结果为0

**原因**：关键词编码问题或页面加载超时

**解决**：
- 尝试更换关键词
- 增大 `MIN_DELAY` 和 `MAX_DELAY`
- 检查网络连接

### 4. 程序运行缓慢

**解决**：
- 减小 `MIN_DELAY` 和 `MAX_DELAY`（但不建议低于0.3秒）
- 减小 `RATE_LIMIT_DELAY_MIN` 和 `RATE_LIMIT_DELAY_MAX`
- 增大 `RATE_LIMIT_THRESHOLD`

### 5. 浏览器初始化失败

**解决**：
- 确认已安装 Edge 或 Chrome 浏览器
- 关闭所有浏览器窗口后重试
- 检查浏览器路径是否正确
